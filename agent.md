# agent.md

## 게임 제작 규칙

1. 새 게임은 반드시 `games/<game-name>/` 폴더 안에 만든다.
2. 각 게임 폴더의 시작 파일은 `index.html`로 통일한다.
3. 새 게임을 추가하면 메인 페이지(`index.html`)의 게임 카드 섹션에 링크를 함께 추가한다.
4. 메인 페이지에서 접근 불가능한 게임 폴더는 배포 대상에서 제외한다.
