# NIGHT COURIER: Black File

한 줄 소개: 감시망을 뚫고 4개 구역의 단말을 해킹해 금지 문서를 회수하는 싱글 플레이 잠입 액션.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000/games/nightcourier/index.html` 접속

## 조작

- 이동: `W A S D`
- 해킹: `E` (단말 근처에서 유지)
- 미끼 배치: `Space`
- 스캔: `Q` (쿨다운)
- 집중 모드(시간 압축 이동): `Shift`
- 일시정지: `Esc`

## 시스템

- 타이틀 화면 + Continue / New Run / Options / Credits
- 옵션: 난이도, 볼륨/음소거, 화면 흔들림 감소, 탐지 보조, 텍스트 크기
- 4챕터 진행(구역별 다른 적 구성/환경/압박)
- 챕터 클리어 강화 선택(해킹 속도/이동 속도/미끼 확장)
- 승리/패배 화면, 즉시 재도전, 메인 메뉴 복귀
- 로컬 저장: 옵션, 최고 랭크/최단 기록, 챕터 체크포인트 Continue

## 플레이 목표

- 각 챕터의 모든 단말 해킹 후 출구로 탈출
- HP(무결성) 0이 되면 실패
- 시간 압박과 탐지를 관리해 최종 챕터까지 완주
