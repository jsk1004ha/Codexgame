# SUNSHARD: Bastion Rite

한 줄 소개: 코어를 지키며 4개 섹터 침공과 최종 보스까지 돌파하는 싱글 플레이 전술 생존 액션.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000/games/sunshard/index.html` 접속

## 조작

- 이동: `WASD` 또는 방향키
- 기본 공격: `Space`
- 대시(짧은 무적): `Shift`
- 시길 펄스(광역 넉백): `E`
- 일시정지/재개: `Esc`
- 음소거 토글: `M`

## 시스템 요약

- 4개 섹터 + 최종 보스의 15분+ 진행 구조
- 섹터 진입 시 무료 강화와 전장 변화
- 적 유형 5종 + 보스 1종 패턴 단계 변화
- 실패/승리 결산 화면, 즉시 재도전 루프
- 로컬 저장: 옵션, 최고 점수, Continue 가능 여부
- 접근성 옵션: 볼륨, 스크린셰이크, 고대비, 난이도
