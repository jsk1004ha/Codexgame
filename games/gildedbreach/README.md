# GILDED BREACH: House of Risk

15분 작전형 싱글 플레이 전술 액션 웹게임입니다. 도박 성채를 돌파하며 위험 배당 업그레이드를 선택해 최종 보스 `HOUSE PRIME`를 격파하세요.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 아래 주소 접속
   - 허브: `http://localhost:8000/`
   - 게임 직접 실행: `http://localhost:8000/games/gildedbreach/`

## 조작

- 이동: `WASD` 또는 `방향키`
- 공격: `J` 또는 `마우스 좌클릭(홀드 가능)`
- 대시(무적 회피): `K` 또는 `Shift`
- 리스크 펄스(광역 기술): `L`
- 일시정지/재개: `Esc`

## 게임 구조 (15분 기준)

- **구간 I (15:00~11:30)**: 로비 돌파, 기본 전투와 회피 학습
- **구간 II (11:30~7:30)**: 테이블 홀, 지뢰/볼트 위험과 업그레이드 선택 본격화
- **구간 III (7:30~3:00)**: 볼트 링, 고위협 적 조합 + 자원 압박
- **구간 IV (3:00~0:00)**: 하우스 코어, 보스 `HOUSE PRIME` 클라이맥스

## 핵심 특징

- 4개 구간 진행, 보상 선택, 보스전 포함의 완결 루프
- 타이밍(대시), 위치 선정, 자원(스태미나/칩) 운영, 위험 감수 선택이 동시에 작동
- 옵션 제공: 난이도, 음소거/볼륨, 저자극 모드, 큰 글씨 HUD, 보조 조준
- 로컬 저장: 최고 점수/클리어 수/최단 클리어, 작전 중단 시 이어하기
- 외부 라이브러리/에셋 없이 `HTML + CSS + Vanilla JS + Web Audio API`만 사용
