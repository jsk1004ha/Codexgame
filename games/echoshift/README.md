# ECHO SHIFT: Night Circuit

네온 전장을 돌파해 코어를 안정화하고 최종 보스 **루인 센티넬**을 격파하는 싱글 플레이 액션 게임입니다.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 아래 주소 열기
   - 허브: `http://localhost:8000/`
   - 게임 직행: `http://localhost:8000/games/echoshift/`

## 조작

- 이동: `WASD` 또는 `방향키`
- 기본 공격: 자동 추적 사격
- 대시: `Shift` (에너지 35% 소모)
- 펄스 방출: `Space` (에너지 55% 소모, 근거리 정리)
- 일시정지/재개: `P` 또는 `ESC`

## 게임 구조

- **구역 I**: 드론 처치로 전투 감각 학습
- **구역 II**: 데이터 노드 회수 중심 목표전
- **구역 III**: 장기 생존/탄막 대응 구간
- **최종전**: 보스 탄막 패턴 파훼

## 시스템

- 타이틀, 옵션, 일시정지, 결과 화면
- 난이도(어시스트/노멀/하드)
- 볼륨/음소거/화면 흔들림/플래시 토글
- 강화 선택(구역 전환 시 3지선다)
- 최고 점수 로컬 저장(localStorage)
- Web Audio API 기반 실시간 효과음
