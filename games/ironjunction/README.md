# IRON JUNCTION: Signalmaster

도시 철도망 관제실에서 분기 스위치와 진입 신호를 조작해 4개 챕터를 완수하는 싱글 플레이 실시간 전략 게임.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행:
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 `http://localhost:8000/games/ironjunction/` 접속.
3. `New Run`으로 시작.

## 조작

- `1 / 2 / 3`: 각 분기 스위치 토글
- `Q / W / E`: 라인 진입 신호 정지/해제
- `Space`: Focus Slow(시간 감속)
- `P` 또는 `Esc`: 일시정지
- 마우스 클릭: 캔버스 내 분기 스위치 직접 토글

## 핵심 특징

- **15분+ 완주 구조**: 4개 챕터(각 4~5분)로 목표 수송량과 압박이 단계적으로 상승.
- **구간 변화**: 반대편 열차, 고속 열차, 위협 폭주 구간 등 챕터별 규칙 변화.
- **의미 있는 선택**: 챕터 사이 강화 1개 선택으로 운영 방식 변화.
- **명확한 승패 조건**: 무결성, 위협 지수, 수송량 달성 여부로 결과 결정.
- **접근성/옵션**: 난이도, 볼륨/음소거, 연출 감소, 고대비, 텍스트 크기 제공.
- **저장 요소**: 로컬 최고 점수 및 옵션 자동 저장.
