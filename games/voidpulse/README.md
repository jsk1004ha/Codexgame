# VOIDPULSE: Last Beacon

네온 폭풍 아레나에서 비콘을 지키며 충전 100% 후 등장하는 보스를 격파하는 싱글 플레이 액션 게임입니다.

## 실행 방법

1. 저장소 루트에서 정적 서버를 실행합니다.
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 아래 주소를 엽니다.
   - `http://localhost:8000/games/voidpulse/index.html`

오프라인 환경에서는 `games/voidpulse/index.html` 파일을 직접 열어도 플레이할 수 있습니다.

## 조작 방법

- 이동: `WASD` 또는 `방향키`
- 파동 공격: `Space`
- 대시 + 강화 파동: `Shift`
- 일시정지/재개: `Esc` 또는 `P`
- 즉시 재시작: `R`
- 음소거 토글: `M`

## 게임 규칙

- 승리: 충전 100% 이후 등장하는 보스 **Oblivion Eye** 처치
- 패배: 플레이어 HP 0 또는 비콘 내구도 0
- 반복 플레이 요소:
  - 최고 점수 저장
  - 클리어 시 `버스트 파동` 해금
  - 보조 모드/효과 옵션으로 자기 스타일에 맞춰 도전

## 접근성/옵션

- 마스터 볼륨 조절
- 화면 흔들림 On/Off
- 강한 플래시 효과 On/Off
- 보조 모드(피해 완화, 충전 속도 증가)

모든 설정과 기록은 `localStorage`에 저장됩니다.
