# PRISMSHIFT: Terminal Run

붕괴 중인 균열 구역에서 12분 생존 + 보스 처치를 노리는 싱글 플레이 타임 액션 슈터입니다.

## 한 줄 소개

시간을 감아 되돌리는 `Fracture Pulse`로 전황을 뒤집으며 3개 섹터와 최종 보스를 돌파하세요.

## 실행 방법

1. 저장소 루트에서 정적 서버를 실행합니다.
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 접속합니다.
   - 허브: `http://localhost:8000`
   - 직접 실행: `http://localhost:8000/games/prismshift/index.html`

## 조작 방법

- 이동: `WASD` 또는 `Arrow`
- 사격: `마우스 좌클릭 홀드` 또는 `Space`
- 대시: `Shift`
- 되감기 파동: `E` (Thread 35% 필요)
- 일시정지: `Esc`

## 주요 특징

- **완결 루프**: 시작 → 3개 섹터 진입 → 최종 보스전 → 클리어/실패 결산.
- **10분+ 진행 변화**: 3:30, 7:00 구간 전환 + 10:00 보스 출현 + 12:00 제한.
- **선택적 숙련 요소**: 대시 무적 프레임, 되감기 타이밍, 위험 지대/투사체 회피, 리소스 루팅.
- **필수 시스템 완비**: 타이틀, 옵션, 크레딧, 일시정지, 재시작, 승/패 화면, 로컬 최고 기록/이어하기.
- **접근성 옵션**: 난이도, 모션 효과 토글, HUD 크기, 마스터 볼륨/음소거.
- **오디오 피드백**: Web Audio API로 공격/피격/회피/획득/승리/실패/UI 효과음 제공.
