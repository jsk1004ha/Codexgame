# RIFT WARDEN: EMBER OATH

균열 요새의 코어를 지키며 3단계 침공과 최종 보스전을 돌파하는 싱글 플레이 전술 아레나 액션 게임입니다.

## 실행 방법

1. 저장소 루트에서 정적 서버를 실행합니다.
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 아래 주소로 접속합니다.
   - 허브: `http://localhost:8000`
   - 직접 실행: `http://localhost:8000/games/riftwarden/index.html`

## 조작

- 이동: `WASD` 또는 `Arrow`
- 조준: `마우스`
- 사격: `좌클릭 홀드` 또는 `Space`
- 대시: `Shift`
- 펄스 충격파: `E`
- 일시정지: `Esc`

## 주요 시스템

- **4단계 진행 구조**: Breach(초반) → Storm(중반) → Siege(후반) → Tyrant(보스전).
- **중간 성장 선택**: 2분, 6분, 9분 시점마다 전술 업그레이드 선택.
- **명확한 승/패 조건**: 코어 또는 플레이어 쉴드 파괴 시 패배, 보스 처치 시 승리.
- **옵션/접근성**: 난이도, 코어 재생 보조, Reduced Flash, HUD 스케일, 볼륨/음소거.
- **저장 요소**: 최고 점수, 최고 생존 시간, 클리어 횟수, 이어하기(진행 스냅샷).

## 팁

- 고열(Heat)이 높으면 화력이 떨어집니다. 대시와 냉각 픽업으로 열을 관리하세요.
- 근접 위협은 `E` 펄스로 밀어내고, 코어 주변 위험 지대를 먼저 정리하면 안정적입니다.
