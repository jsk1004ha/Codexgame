# STARFORGE: Ember Raid

한 줄 소개: 자동 사격 로그라이트 생존전에서 코어를 모아 보스 함선을 격파하는 5~10분 싱글 플레이 캔버스 액션 게임.

## 실행 방법

1. 저장소 루트에서 정적 서버 실행:
   ```bash
   python3 -m http.server 8000
   ```
2. 브라우저에서 아래 주소 접속:
   - 허브: `http://localhost:8000/`
   - 게임 직행: `http://localhost:8000/games/starforge/`

## 조작 방법

- 이동: `WASD` 또는 `←↑↓→`
- 대시(짧은 무적): `Space`
- 충격파(근접 광역): `Shift`
- 일시정지/해제: `P`
- 옵션 열기: `O`
- 사격: 자동

## 게임 목표

- 일반 페이즈에서 드론을 파괴하고 **에너지 코어 20개**를 수집합니다.
- 코어를 모으면 보스 **Oblivion Carrier**가 등장합니다.
- 보스를 격파하면 클리어, 체력이 0이 되면 게임오버입니다.

## 옵션/접근성

- 마스터 볼륨, 음소거
- 화면 흔들림 줄이기
- 자동 조준 보정(초보자 보조)
- 난이도: 쉬움/보통/어려움(어려움은 1회 클리어 후 해금)

## 저장 데이터(localStorage)

- 최고 점수
- 하드 모드 해금 상태
- 옵션 설정값
